## 📦 요약 문서에 포함된 내용

각 요약 문서는 다음을 포함합니다:

1. **모든 마크다운 셀** - 설명, 구조, 이론적 배경
2. **모든 코드 셀** - 전체 로직, 함수 정의, 데이터 처리
3. **실행 결과** - print 출력, 데이터프레임, 통계량
4. **그래프 정보** - 차트/그래프 출력 표시 (이미지 데이터 제외)

## 🔄 재생성 방법

노트북이 업데이트되면 요약 문서를 재생성:

```bash
python scripts/create_notebook_summary.py
```

```bash

# 상세 비교 (전체 내용 추출 및 비교)
```

## 📝 생성 원칙

요약 문서는 다음 원칙으로 생성됩니다:

1. **완전성**: 모든 코드와 마크다운 포함
